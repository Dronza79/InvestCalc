# 📈 InvestCalc & Portfolio Balancer

**InvestCalc** — это профессиональный инструмент на Python для финансового моделирования и управления портфелем. Приложение объединяет в себе умный калькулятор сложных процентов с учетом инфляции и налогов, а также модуль ребалансировки активов.

---

## ✨ Основные возможности

### 1. Модуль «План инвестиций»
Программа обладает встроенной логикой **искомого параметра**. Заполните любые поля, и калькулятор сам поймет, что нужно рассчитать:
*   **Капитал** 
*   **Регулярный платеж** 
*   **Начальная сумма** 
*   **Инвест. горизонт** 
*   **Годовая доходность** 

**Продвинутые настройки расчета:**
*   **Инфляция:** Учет среднего индекса 8% для оценки реальной стоимости денег.
*   **Налоги:** Расчет прогрессивного НДФЛ на инвестиционный доход.
*   **Гибкий график:** Раздельная настройка периодичности платежей и капитализации (от ежедневной до ежегодной).
*   **Кратность:** Округление результатов для удобства (1, 100, 500, 1000).

### 2. Модуль «Баланс портфеля»
Инструмент для контроля долей активов: **Акции, Облигации, Фонды, Металлы**.
*   **Режим докупки:** Расчет сумм, необходимых для достижения заданного баланса.
*   **Режим частичного пополнения:** Оптимальное распределение активов с учетом пополнения.
*   **Валидация:** Проверка общей доли (не более 100%) и автозаполнение текущего капитала.

---

## 🖥 Интерфейс
Приложение имеет двухпанельную структуру на базе **PySimpleGUI**:
1.  **Левая панель:** Ввод данных и выбор параметров (чекбоксы налогов, инфляции, выпадающие списки периодов).
2.  **Правая панель:** Вывод результатов в трех вкладках:
    *   📄 **Отчет:** Текстовое резюме со всеми расчетами.
    *   📊 **График:** Визуализация роста капитала. При этом график интерактивный.
    *   📅 **Таблица:** Пошаговая динамика изменений по годам.

## ⚙️ Установка и запуск

### Запуск из исходного кода
1. Установите Python 3.11+.
2. Клонируйте репозиторий.
3. Установите зависимости:
```bash
pip install -r req.txt
```
4. Запустите приложение:
```bash
python main.py
```

### Сборка исполняемого файла (.exe)
Для создания автономного файла для Windows 10+ выполните:
```bash
pyinstaller -F --windowed -i favicon.ico -n investcalc main.py
```
*Результат будет находиться в папке `dist/`.*

## ⚠️ Дисклеймер
Приложение предназначено для ознакомительных расчетов и не является индивидуальной инвестиционной рекомендацией. Расчет инфляции и налогов носит прогнозный характер.
